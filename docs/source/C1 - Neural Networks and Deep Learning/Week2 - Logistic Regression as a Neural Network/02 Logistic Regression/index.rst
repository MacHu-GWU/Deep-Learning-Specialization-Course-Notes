Logistic Regression
==============================================================================
逻辑回归是用于二值分类的情景，二值分类时我们更多的是想知道输出为1的概率值，通常情况下大于50%我们就认为是1，小于50%就认为是0。对比线性回归，用一个向量积+一个常数得到的结果通常是一个数值，可能是负，也可能非常大，这个结果很不适合用来表示概率。

所以逻辑回归的本质就是在线性回归的基础上再加上一个sigma函数，将 (-∞, +∞) 这个区间映射到 (0, 1)这个区间上。

::

                                  1
    σ(wT * x + b) = σ(z) = ---------------
                            1 + e ** (-z)

            |/
